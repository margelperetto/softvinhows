Back-end softvinho
Para acessar algum recurso, adicione o path /api/nome_do_recurso


Recursos disponíveis:
/cliente
/vinho
/venda


Operações:

/save/{JSON}
Retorno: JSON com id

/delete/{num_id}
Retorno:true ou false

/get/{num_id}
Retorno: JSON completo do objeto

Nenhum recurso adicionado ao path: lista todoa em formato JSON


Exemplo para listar todos os clientes:
http://localhost:8080/softvinhows/api/cliente

Exemplo para buscar o cliente de código 1:
http://localhost:8080/softvinhows/api/cliente/get/1
